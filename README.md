# Python Bank of Joe

A simple, clean Bank Management System written in Python using OOP principles.  
This project includes:

* Create & manage accounts  
* Deposit & withdraw (daily withdrawal limit included)  
* Interest calculation  
* Transfer between accounts  
* Account lookup  
* List all accounts  
* Terminal-based menu  

##  Features

- **Account class**
  - Daily withdrawal limit
  - Interest application
  - Deposit & withdraw with validations
  - Tracks remaining daily limit

- **Bank class**
  - Open/close accounts
  - Transfer amounts
  - Lookup & list accounts
  - Auto-incrementing account numbers

- **Application**
  - Menu-driven interface
  - Error handling
  - Clean user prompts
